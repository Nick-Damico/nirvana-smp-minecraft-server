Nirvana-SMP:PS4

Welcome to Nirvana-SMP: PS4 Server!

Nirvana-SMP, survival multiplayer, is a Playstation 4 server hosted in East Tennessee with a great Community of players who love Minecraft and want to share their experience with players like them.

Our goal is to provide the best Minecraft multiplayer experience possible on the PS4. We accomplish this by bringing together creative, friendly and supportive minecraft players that help us build the best community possible.

Ready to Start Crafting? Get Whitelisted Here.

Server Beta testing starting… Feb.

Jan. 13th, 2015 by NickD28

The time to start testing Nirvana-SMP server is coming up quick, and our major concerns going into the beta testing are stability, connectivity, and latancy. Opening day will be Wednesday Feb. 4th followed by Saturday Feb. 7th starting @5am EST.

During this beta test the server will have intermitten hours of uptime. We are going to start with two 24hr days for the first week or two, and more days will be added as we feel that we are able to properly handle server load.

I will be going through the whitelist applications and adding approved players to my friends list, but there is going to be a cut off day for this beta test which will be Jan. Sunday 25th, so if you are wanting to get in on the beta run make sure to complete and submit the whitelist app by then. Obviously since my account will need to be logged in at all times for the server to run that will leave 7 of the 8 player spots availiable so please take that into consideration when trying to join.

Also if you are going to be bothered by possible interrupted gameplay and other unforseen problems then please sit out this initial test until we can get a handle on any problems that will come up.

I'm really excited to get this server up and I look forward to playing with everyone in Feb.
